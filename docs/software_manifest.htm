Open Source Build and Download Information

 

HALCoGen offers the configuration of an open source operating system named FreeRTOS. The original source can be obtained from the FreeRTOS home page (http://www.freertos.org/).

 

HALCoGen uses the open source JavaScript engine SpiderMonkey from Mozilla internally. The original source can be obtained from the Mozilla (home page: http://www.mozilla.org/js/spidermonkey/) SipderMonkey ftp server (ftp://ftp.mozilla.org/pub/mozilla.org/js/js-1.8.0-rc1.tar.gz).

 

The SpiderMonkey source as used in HALCoGen can be obtained together with an Visual Studio 2010 Project to rebuild SpiderMonkey from an TI ftp server (ftp://ftp.ti.com/pub/tms470/HCG/TIJS32.zip ).

 

To rebuild the Microsoft Visual Studio 2010 Project for SpiderMonkey, please download it from the Texas Instruments ftp server (ftp://ftp.ti.com/pub/tms470/HCG/TIJS32.zip ), open the Microsoft Visual Studio 2010 project file TIJS32.vcxproj with Microsoft Visual Studio 2010 and press the build button to create the Dll: TIJS32.dll.
